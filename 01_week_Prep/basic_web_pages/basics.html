Headings

Headings define the outline of your site. There are six levels of headings.

Lists

This is how you make an unordered list.

  • Add an "ul" element (e.g. unordered list)
  • Add each item in its own "li" element
  • Since this is unordered, the order is not specific

Practical uses: item inventories; product features

Ordered Lists

This is how you make an ordered list.

  1. Add an "ol" element (e.g. ordered list)
  2. Add each item in its own "li" element
  3. Order is key here

Practical uses: recipes; instructions; tables of content
